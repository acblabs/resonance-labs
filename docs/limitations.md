# Limitations

ResonanceLab Phase 1 does not estimate fill level.

Known limitations:

- Browser audio processing can distort amplitude, decay, and spectral features.
- Device speaker-to-microphone bleed may dominate the object response.
- Mobile browser behavior is not verified until real devices are tested.
- The API currently returns basic WAV metrics and placeholder alignment metadata.
- Accuracy claims require calibrated datasets and Phase 2+ DSP validation.
